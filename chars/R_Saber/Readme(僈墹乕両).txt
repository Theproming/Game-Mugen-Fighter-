【AIパッチ】
 
【AI説明】

　対応キャラ：deadhead氏作成 β版 セイバー(Fate/EXTRA)
　　　　　　　　　　　
【AI設定】

  ・AIの起動が遅い場合はR_Saber.cmdファイルの
　　　　　　　　　　;trigger66 = 1の行にある一番前の;を消してAIを常時起動にしてください

　・動画使用やプレイヤーでAIに挑戦する場合は常時起動するのがよいと思われます

　※プレイヤーが使用する場合は必ず;をつけたままにしてください
　　　　　　　　　　　　　　　　　　　　　　　　　　　  常時起動だと操作出来なくなります

　※戦闘前のイントロは飛ばすとAIの起動が遅くなりますので飛ばさないようお願いします
                                        最悪１試合まるまるAIが起動しないことがありえます

                                           :
                                           :
  trigger60 = Command = "AI59"
  trigger61 = Command = "AI60"
  trigger62 = Command = "AI60"
  trigger63 = Command = "AI62"
  trigger64 = Command = "AI63"
  trigger65 = Helper(10000),var(50)
  ;trigger66 = 1                    ;AI常時起動 ←この行の先頭のセミコロン(;)を削除でAI常時起動
  var(50) = 10
  ;add_end

【AI設定方法】

　AIのファイルをR_Saberフォルダに上書きしてください
  ※念の為、適応する前に本体のバックアップと取ることをオススメします
   
　MUGEN本体のselect.defファイルのrandomselectをR_Saber/R_Saber_AI.defと変更してください
